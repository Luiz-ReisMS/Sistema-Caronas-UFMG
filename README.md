#SISTEMA DE CARONAS INTELIGENTE - UFMG  
**Disciplina:** Programação e Desenvolvimento de Software II (PDS2) - 2026.1  
**Tema:** Sistema de apoio à mobilidade acadêmica para alunos oferecerem e solicitarem caronas.  
  
##Escopo e Funcionalidades (Entrega E2)  
Um sistema onde alunos oferecem e solicitam caronas.  
*
* Cadastro de usuários (motoristas e passageiros)
* Cadastro de viagens(origem, rota, destino, horário e quabtudade de vagas disponíveis)
* Solicitação de carona
* Aprovação/rejeição pelo motorista
* Sistema de avaliação de motoristas e passageiros(0 a 5)
* Histórico de viagens do motorista
* Cancelamento de viagem
* Fila de espera se número de vagas menor do que número de solicitaçôes da viagem
* Denúncia de má conduta de motoristas e passageiros
